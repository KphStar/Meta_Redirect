Privacy Policy

Effective Date: July 4, 2025

This app ("Threads Desktop by KphStar") uses Meta’s Threads API to allow users to log in and view their own Threads posts.

Information Collected

We only collect your Meta access token (via the OAuth login process) for the sole purpose of retrieving your Threads posts. This token is not stored, shared, or reused after your session ends.

Data Usage

All retrieved data is used solely on your local machine and is visible only to you, the authenticated user. We do not transmit any data to external servers.

Third-Party Services

This app uses Meta’s Threads API for authentication and data access. Your data is handled according to Meta’s own privacy policy.

Your Consent

By using this app, you consent to the limited use of your Threads data as described above.
